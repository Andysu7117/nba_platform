# NBA Platform

Full-stack NBA analytics platform — a **FastAPI** backend and a
**React + TypeScript + Vite** frontend for game prediction, playoff simulation,
standings, schedules and player stats.

The application lives in [`nba-playoff-predictor/`](./nba-playoff-predictor) —
see its [README](./nba-playoff-predictor/README.md) for architecture and run
instructions.

`NBA Platform.dc.html` is the original Claude Design source for the "Hardwood"
UI, now implemented as the modular React frontend.
